# 全国农产品可视化系统 python760

## 项目概述

全国农产品可视化系统是一款基于 Python 语言开发的大作业项目，主要用于农产品的数据采集、存储、展示和分析，以及提供二十四节气时令推荐和智能食谱助手功能。

## 技术栈

- 爬虫技术
- PyCharts 绘图
- 网络编程
- MySQL 数据库

## 功能模块

### 数据采集与存储

- 将数据存储到 MySQL 数据库中。

### 数据可视化展示

- 利用 PyECharts 绘制四种图表对数据分析。

### 二十四节气时令推荐

- 根据节气推荐应季蔬菜。
- 提供相关的营养价值说明。

### 智能食谱助手

- 根据用户输入的需求，推荐合适的食谱。
- 提供详细的烹饪步骤和营养分析。

## 系统角色

- 系统管理员：负责数据的维护和系统的日常监控。
- 普通用户：查看农产品数据、接受时令推荐、使用智能食谱助手。

## 运行环境

- Python 环境推荐版本：3.6 或以上
- MySQL 数据库版本：5.7 或以上
- 操作系统：支持所有主流操作系统，如 Linux、Windows、macOS

## 部署步骤

1. 安装 Python 环境。
2. 安装依赖库：爬虫相关库、PyCharts、数据库连接库等。
3. 配置 MySQL 数据库。
4. 导入数据模型与数据。
5. 部署应用服务器。

## 目录结构

```
/
├── app           # 应用程序主目录
│   ├── models    # 数据模型目录
│   ├── views     # 视图层目录
│   ├── controllers   # 控制器目录
│   └── static    # 静态资源目录
├── config        # 配置文件目录
├── database      # 数据库相关文件目录
└── tests         # 测试代码目录
```
